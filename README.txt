Demo use of Terminal-IDE system from:

		https://play.google.com/store/apps/details?id=com.spartacusrex.spartacuside

to build a demo, "Busy Box", type Android program.

The Terminal-IDE system lets you edit and build directly on your
Android device, without requiring another computer.

This demo has some notes about running the environment in overview.html,
and has a Makefile to replace the scripts that are in the T-IDE demos.
